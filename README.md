## Web3 Asset Dashboard

A Web3-enabled application that allows users to log in via MetaMask and view on-chain asset information.

### Features
- Wallet authentication using MetaMask
- ETH balance retrieval from Ethereum testnet
- Recent transaction history display
- Responsive dashboard UI

### Tech Stack
- Next.js
- ethers.js
- Tailwind CSS
- Ethereum (Sepolia Testnet)
